## GitHub contributor graph

The [**Contributors**](https://github.com/MalikZeeshan1122/AI-Scientist/graphs/contributors) tab on GitHub is generated automatically from **commits merged into the default branch**, using the author email linked to each GitHub account.

To appear there:

1. Commit with an email [verified on your GitHub account](https://docs.github.com/en/account-and-profile/setting-up-and-managing-your-personal-account-on-github/managing-email-preferences/adding-an-email-address-to-your-github-account).
2. Open a pull request and get it merged (or have a maintainer merge your commits).

This file is an explicit credit list and may include contributors before their first merged commit.
