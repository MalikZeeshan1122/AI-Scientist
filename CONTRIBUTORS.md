## Insights → Pulse → Contributors (why you do not see yourself yet)

That page is **generated only by GitHub** from **git commit authors** on the default branch (`main`). You **cannot** add your name there through `CONTRIBUTORS.md`, repo settings, or the GitHub UI—it is not editable.

Pulse often labels the chart **“excluding merge commits”**, so only non-merge commits are plotted in some views.

Right now **`cursoragent`** appears because recent commits were authored with that identity (e.g. Cursor automation). Until **you** create commits where **you** are the author and GitHub can match your email, **your GitHub user will not replace `cursoragent` on that chart**.

### Make your future commits count as you

1. Add and verify your email: [GitHub → Settings → Emails](https://github.com/settings/emails).
2. On the machine where you commit, set git to use that **exact** verified address:

```bash
git config --global user.name "Muhammad Zeeshan"
git config --global user.email "you@example.com"
```

Use the same email GitHub shows as verified (or your GitHub **noreply** address if you keep email private).

3. Make commits locally yourself (`git commit`) and push to `main` (or merge a PR whose commits list you as author). After they land on `main`, Pulse → Contributors updates within about a day.

### Optional: note collaboration on a commit

You can add [Co-authored-by](https://docs.github.com/en/pull-requests/committing-changes-to-your-project/creating-and-editing-commits/creating-a-commit-with-multiple-authors) trailers so multiple people appear on the commit page; **Pulse still relies primarily on the primary commit author** for the contributor chart—keep `user.email` correct for your own contributions.
